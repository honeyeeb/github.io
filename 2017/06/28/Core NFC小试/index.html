最近北京地铁在房山线试运行手机刷卡，实际上就是手机支持NFC功能的方便了，鉴于市场上主流的手机都支持全功能NFC，以后出门忘记带乘车卡也可以坐地铁了，不过iPhone用户此时比较心塞，由于iPhone安全策略一直没有开放NFC功能提供开发者使用。BUT，WWDC17带来了众多功能中的NFC功能，本文在iOS11 beta版本上做一个小小的测试，看看功能如何。
